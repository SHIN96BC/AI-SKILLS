# 웹 디자인 원칙 레퍼런스

## 1. 사용성 원칙

### 3초 룰
- 페이지 진입 후 3초 이내에 "이 페이지가 뭘 하는 곳인지" 파악 가능해야 함
- 히어로 영역의 헤드라인이 핵심 가치를 즉시 전달
- 시각적 계층 구조로 시선 흐름 유도: 헤드라인 → 서브텍스트 → CTA

### 최소 클릭 원칙
- 핵심 기능까지 최대 3클릭 이내 도달
- 네비게이션은 최대 2depth까지 (메가메뉴 허용 시 예외)
- CTA는 스크롤 없이 보이는 위치에 최소 1개 배치

### 일관성
- 같은 종류의 버튼은 같은 스타일
- 같은 인터랙션은 같은 피드백 (호버, 클릭, 포커스)
- 간격과 여백은 8px 기반 그리드 시스템 사용 (8, 16, 24, 32, 48, 64)

### 피드백
- 모든 클릭 가능한 요소에 호버/포커스 상태 제공
- 폼 제출 후 성공/실패 피드백 즉시 표시
- 로딩 상태는 스켈레톤 UI 또는 스피너로 표현

## 2. 레이아웃 원칙

### 여백과 밀도
- 콘텐츠 간 여백은 넉넉하게 (최소 24px)
- 섹션 간 간격: PC 80~120px, 모바일 48~64px
- 텍스트 줄 길이: 최대 75자 (가독성 최적)
- 문단 간격: 줄 높이의 1.5~2배

### 그리드 시스템
- PC: 12열 그리드, 거터 24px, 최대 너비 1200~1440px
- 태블릿: 8열 그리드, 거터 16px
- 모바일: 4열 그리드, 거터 16px, 좌우 패딩 16~20px

### 시각적 계층
- 크기, 굵기, 색상, 여백으로 중요도 표현
- 타이포그래피 스케일: h1 > h2 > h3 > body > caption
- 주요 액션은 채워진 버튼, 보조 액션은 아웃라인 버튼

## 3. 반응형 설계 원칙 (Mobile First)

### 브레이크포인트
| 구간 | 너비 | Tailwind 접두사 | 설명 |
|------|------|----------------|------|
| 모바일 | ~639px | 기본값 | 기준 레이아웃 |
| 큰 모바일 | ≥ 640px | sm: | 소형 태블릿 |
| 태블릿 | ≥ 768px | md: | 태블릿 세로 |
| 데스크톱 | ≥ 1024px | lg: | 일반 PC |
| 큰 데스크톱 | ≥ 1280px | xl: | 큰 모니터 |
| 와이드 | ≥ 1536px | 2xl: | 와이드 모니터 |

### 반응형 변환 패턴 (모바일 → PC 확장)
- **그리드 확장**: 1열 → 2열 (md:) → 3~4열 (lg:)
- **네비게이션 전환**: 햄버거 메뉴 → 가로 메뉴 (md:)
- **사이드바 전환**: 드로어/오버레이 → 옆에 고정 (lg:)
- **테이블 전환**: 카드 리스트 → 표 (md:)
- **이미지 비율**: 정방형 → 가로형 (md:)
- **폰트 확장**: 기본 → md: 110% → lg: 120%
- **여백 확장**: 최소 → md: 중간 → lg: 넉넉

## 4. 접근성 체크리스트

- [ ] 색상 대비 비율 4.5:1 이상 (WCAG AA)
- [ ] 모든 이미지에 alt 텍스트
- [ ] 키보드만으로 모든 기능 사용 가능
- [ ] 포커스 인디케이터 명확하게 표시
- [ ] 폼 요소에 label 연결
- [ ] 터치 타겟 최소 44x44px (모바일)
- [ ] 스크린리더로 콘텐츠 순서가 논리적
- [ ] 움직이는 요소에 일시정지 기능 또는 reduced-motion 대응

## 5. 페이지 유형별 설계 패턴

### 랜딩 페이지
- Hero → 가치 제안 → 기능 소개 → 사회적 증거 → CTA
- 스크롤 당 하나의 메시지에 집중
- CTA는 2~3회 반복 배치

### 대시보드
- 좌측 사이드 네비게이션 + 상단 헤더
- 핵심 지표를 최상단에 카드로 배치
- 데이터 테이블 + 필터/검색 조합

### 소개/기업 사이트
- 브랜드 스토리 중심의 시각적 구성
- 큰 이미지/영상 활용
- 신뢰 요소(실적, 파트너, 인증) 배치

### 이커머스
- 상품 그리드 + 필터 사이드바
- 상품 상세: 이미지 갤러리 + 정보 + CTA
- 장바구니/결제 흐름 최소화

### 블로그/콘텐츠
- 최대 너비 제한된 본문 영역 (max-width: 720px)
- 사이드바 선택적 (ToC, 관련 글)
- 타이포그래피 중심 레이아웃
